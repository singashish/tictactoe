# Tic Tac Toe Game

This is a simple command-line-based Tic Tac Toe game implemented in C++.

## Prerequisites

- CMake: Make sure you have CMake installed to configure and build the project.
- MinGW: You need the MinGW development environment to compile the project with g++.

## Getting Started

1. Clone this repository to your local machine.

2. Navigate to the project directory.

3. Run the `build.bat` script to build and run the game.

## How to Play

- The game starts with a random choice of first player (you or the bot).
- Players take turns to make their moves.
- To make a move, enter the position (1-9) where you want to place your symbol.
- The first player to form a line (horizontal, vertical, or diagonal) of their symbols wins.
- If the entire board is filled without a winner, the game ends in a draw.

Enjoy playing Tic Tac Toe!
